# DOCAI

## Description

DOCAI is a tool that can read documentation in the MD format and answer questions based on it. It is designed to make it easier for users to access and interact with documentation using natural language queries.

## Installation

To install DOCAI, follow these steps:

1. Clone the repository: `git clone https://github.com/Izobov/docai.git`
2. Navigate to the project directory: `cd docai`
3. Install dependencies: `yarn install`

## Usage

DOCAI can be used in two ways:

### 1. Reading Documentation

To read the documentation, follow these steps:

1. Launch the DOCAI documentation reader: `yarn doc`
2. Answer the questions in console.

### 2. Chatting with DOCAI

To chat with DOCAI, follow these steps:

1. Launch the DOCAI chat interface: `yarn chat`
2. Start a conversation with DOCAI by typing in your queries or questions.
3. DOCAI will analyze your queries and provide answers based on the available documentation.
